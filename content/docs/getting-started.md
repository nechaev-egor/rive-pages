# Getting Started

Welcome to the Rive Animation Tester documentation.

## Quick Start

1. Open the [tester](/test) page
2. Load an animation by URL or select a preset
3. Use the controls to switch State Machines, Artboards, and ViewModels

## Loading Animations

You can load Rive animations from:

- **URL** — Paste a direct link to a `.riv` file (e.g. from [cdn.rive.app](https://cdn.rive.app))
- **Local file** — Upload a `.riv` file from your device
- **Presets** — Use built-in examples like Vehicles (Bumpy)

## Fit Modes

The viewer supports different fit modes:

- **Contain** — Scale to fit, preserve aspect ratio
- **Layout** — Responsive layout (for files with Layouts)
- **Cover** — Fill container, may crop
- **FitWidth / FitHeight** — Scale to match one dimension
